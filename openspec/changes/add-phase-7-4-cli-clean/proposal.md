## Why
Provide guided flow combining analyze, plan, and optional apply for structural cleanup findings.

## What Changes
- Run analyzers and filter findings for duplicate_block, large_file, and long_function.
- Let user choose one or a small subset, generate plans, and prompt to apply now or store.
- Offer a user-friendly wrapper around existing commands.

## Impact
- Simplifies cleanup selection for common categories.
- Reduces manual command chaining for users.
