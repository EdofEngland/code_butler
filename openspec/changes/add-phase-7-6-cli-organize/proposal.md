## Why
Allow users to act on organize_candidate findings via guided file move plans.

## What Changes
- Run organize-seed analyzer and list organize_candidate findings with suggested folders and files.
- Allow user selection of one or more candidates, generate file-move plans, and offer apply or store actions.
- Enforce small move sets and avoid file content edits.

## Impact
- Users can reorganize files safely via guided plans.
- Scope remains small per plan.
